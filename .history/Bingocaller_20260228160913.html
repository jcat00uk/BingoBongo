How to Play

  • You can start a game immediately by clicking Start Game, or first select cards manually for auto-check.
  • To select cards, click Select Cards and choose which cards you want to track. Confirm to save your selection.
  • If you manually selected cards, you can toggle Auto-Check on or off. Auto-Check will automatically detect the first LINE and first FULL HOUSE in the game.
  • Click Next Number to call the next number randomly.
  • Click Undo to undo the last number called.
  • To check a specific card manually, select its code from the Card Selection dropdown. This will show the current win status of that card.
  • Click End Game to finish and reset the game.
  • If the Bingo Lingo text is visible and Text-to-Speech (TTS) is enabled, click/tap it to have the text read aloud.
  • Toggle Sound 🔊/🔇, TTS 🗣️/🚫, and Night Mode 🌞/🌙 using the buttons at the top of the screen.
  • Any questions? Experiment with the controls or ask a fellow player!
